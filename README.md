### Experiment 1 – Linear Regression

Implementation of regression models using a single input feature and multiple input features.

Files:

* `LinearRegression.py`
* `Multiple_LinearRegression.py`

### Experiment 2 – Multivariate Linear Regression

Implementation of multivariate regression where a single input variable predicts multiple outputs.

File:

* `Multivariate_LR_model.py`


### Experiment 3 – Ridge Regression

Implementation of Ridge Regression using both a standard library approach and a matrix-based formulation.

Files:

* `Ridge_regression.py`
* `Ridge_regression_Matrix_method.py`


### Experiment 4 – Support Vector Machine (SVM)

Implementation of linear Support Vector Machine models for binary classification.

Files:

* `Linear_SVM_model.py`
* `SVM_withSupportVector.py`


### Experiment 5 – Naive Bayes

Implementation of Naive Bayes classifier using one and two input features.

Files:

* `1inputNB_model.py`
* `2inputNB_model.py`


### Experiment 6 – K-Means Clustering

Implementation of K-Means clustering algorithm.

File:

* `k_mean_clustering_model.py`
